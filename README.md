## Getting Started

### 1. Install Requirements

#### Development

- [`node` `16.x`](https://nodejs.org/en/)
- [`pnpm` `6.x`](https://pnpm.io/installation)

### 2. Clone Repo

```bash
$ git clone https://github.com/pognetwork/wallet.git && cd wallet
```

### 3. Install JS Dependencies

```bash
$ pnpm install
```

## Development

### Run dev server

```bash
$ pnpm dev
```
